# 비숍

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/1799)

## 사용 알고리즘

- 백트래킹

## 풀이

비숍은 같은 색 칸끼리만 서로 공격하므로 흑/백 칸을 분리해서 각각 최대 개수를 구한 뒤 합쳤다. 각 칸을 순서대로 보면서 비숍을 놓을 수 있으면 비숍을 두는 경우의 수와 두지 않는 경우의 수로 나누어 재귀를 진행하는 방식으로 탐색했다.

## 복잡도

- 시간: 최악 `O(2^(N^2/2))`
- 공간: `O(N^2)`

## 한 줄 회고

색을 분리하면 탐색 공간이 눈에 띄게 줄어든다.
