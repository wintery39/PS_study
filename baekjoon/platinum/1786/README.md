# 찾기

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/1786)

## 사용 알고리즘

- KMP
- 문자열 매칭

## 풀이

패턴의 접두사/접미사 정보를 failure 배열로 만든 뒤, 본문을 한 번만 훑으며 불일치가 생기면 가능한 위치로만 되돌아갔다. 패턴 끝까지 맞으면 시작 위치를 기록하고 다음 후보로 이어 갔다.

## 복잡도

- 시간: `O(|T| + |P|)`
- 공간: `O(|P|)`

## 한 줄 회고

KMP는 "다시 어디서부터 비교할지"를 미리 저장하는 방식이다. 문자열끼리의 비교를 위해 많이 사용되는 방식이다.
