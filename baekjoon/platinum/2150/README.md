# Strongly Connected Component

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/2150)

## 사용 알고리즘

- 코사라주 알고리즘
- DFS

## 풀이

원래 그래프에서 DFS 종료 순서를 쌓고, 간선을 뒤집은 그래프에서 그 역순으로 다시 DFS를 돌려 SCC를 분리했다. 각 SCC 내부를 정렬하고 전체 결과도 정렬해서 출력 형식에 맞췄다.

## 복잡도

- 시간: `O(V + E)`
- 공간: `O(V + E)`

## 한 줄 회고

알고리즘 수업에서 배운대로 종료 순서와 역방향 DFS 조합이 SCC를 가장 직관적으로 나눠 준다.
