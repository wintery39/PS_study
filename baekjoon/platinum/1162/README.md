# 도로포장

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/1162)

## 사용 알고리즘

- 다익스트라

## 풀이

다익스트라 알고리즘을 사용할 떄 `(현재 도시, 남은 포장 횟수)`를 하나의 node로 보았다. edge를 그냥 지나가는 경우와 포장해서 비용 없이 넘어가는 경우를 모두 힙에 넣으면, 최단 경로를 구할 수 있다.

## 복잡도

- 시간: `O(MK log (NK))`
- 공간: `O(NK)`

## 한 줄 회고

node를 도시로만 보는 것이 아니라 남은 포장 횟수를 포함해서 보는 생각이 중요한 역할을 했다고 생각한다.
