# 큐빙

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/5373)

## 사용 알고리즘

- 구현
- 시뮬레이션

## 풀이

6개 면을 3x3 배열로 두고, 회전 명령이 들어올 때 회전하는 면 자체와 인접한 12칸을 함께 갱신했다. 모든 명령을 순서대로 적용한 뒤 윗면 상태를 출력했다.

## 복잡도

- 시간: `O(TN)`
- 공간: `O(1)`

## 한 줄 회고

회전 규칙만 정확히 정리하면 구현 자체는 반복 작업에 가깝다. 중간에 실수를 하면 오류를 찾기 어렵기 때문에 꼼꼼히 구현하는 것이 중요하다.
