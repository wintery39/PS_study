# 책 정렬하기

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/35298)

## 사용 알고리즘

- 그리디

## 풀이

뒤에서부터 목표 번호를 맞춰 가면서, 현재 끝에 목표가 있으면 바로 제거하고 아니면 허용된 회전을 수행해 위치를 바꿨다. 연산 과정을 그대로 기록해 가능한 경우에는 연산 수와 위치를 출력했다.

## 복잡도

- 시간: `O(N^2)`
- 공간: `O(N)`

## 한 줄 회고

시간복잡도보다는 구현이 중요했던 문제로 가장 빠른 코드는 아니였지만 대회에 맞게 가장 빨리 작성할 수 있는 간결한 형식으로 작성하였다.
