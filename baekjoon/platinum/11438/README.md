# LCA 2

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/11438)

## 사용 알고리즘

- 트리
- BFS
- 최소 공통 조상(LCA)

## 풀이

루트부터 BFS로 깊이와 부모를 구하고, 일정 간격의 조상 정보를 따로 저장했다. 질의에서는 두 정점을 같은 높이대로 맞춘 뒤 공통 조상이 갈라지는 지점을 찾아 마지막에 한 단계씩 올려 LCA를 구했다.

## 복잡도

- 시간: `O(N + M log N)`
- 공간: `O(N ^ 2)` (최악의 경우)

## 한 줄 회고

공간 복잡도와 시간 복잡도의 트레이드 오프를 적절히 생각하여 구현하는 것이 중요한 문제였다.
