# 히스토그램에서 가장 큰 직사각형과 쿼리 3

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/35300)

## 사용 알고리즘

- 인터랙티브
- 탐색

## 풀이

현재 확정된 구간과 나머지 후보 구간을 덱으로 관리하면서 질의를 만들어 응답을 해석했다. 응답으로 얻은 `(l, r, h)`를 이용해 정답이 포함된 구간을 점점 좁혀 가고, 마지막에는 후보를 하나씩 확인해 값을 찾는 방식이다.

## 복잡도

- 시간: 질의 횟수 기준 `O(QN)` 이하
- 공간: `O(N)`

## 한 줄 회고

인터랙티브 문제의 경우 시간 복잡도보다 구현과 설게가 중요하다는 것을 알게 되었다.
