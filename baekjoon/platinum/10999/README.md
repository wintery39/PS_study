# 구간 합 구하기 2

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/10999)

## 사용 알고리즘

- 세그먼트 트리
- 구간 갱신

## 풀이

구간 합을 저장하는 트리를 만들고, 갱신이 필요한 구간은 바로 아래로 모두 내려가지 않고 부모 쪽에 반영 여부를 남겨 두었다. 질의나 추가 갱신이 내려올 때 필요한 경로만 실제로 풀어 주는 방식으로 범위 업데이트와 범위 합 조회를 처리했다.

## 복잡도

- 시간: `O((M+K) log N)`
- 공간: `O(N)`

## 한 줄 회고

구간 갱신 문제는 느리게 갱신하는 세그먼트 트리를 사용하는 것이 거의 필수적이다.
