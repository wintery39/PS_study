# 책 페이지

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/1019)

## 사용 알고리즘

- 수학
- 자릿수 분해

## 풀이

각 자릿수마다 `0~9`가 몇 번 등장하는지 따로 계산했다. 현재 자릿수보다 높은 자리에서 완성되는 묶음 수와, 낮은 자리의 나머지를 이용해 해당 자릿수 기여도를 더해 주면 전체를 직접 세지 않고도 답을 구할 수 있다.

## 복잡도

- 시간: `O(log N)`
- 공간: `O(1)`

## 한 줄 회고

자릿수별로 독립적으로 세면 깔끔하게 구할 수 있다. 구현에 시간을 생각보다 많이 쓴 문제여서 미리 알고리즘을 정리하고 그 다음 문제를 푸는것이 얼마나 중요한지 알 수 있었다.
