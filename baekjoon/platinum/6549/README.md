# 히스토그램에서 가장 큰 직사각형

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/6549)

## 사용 알고리즘

- 분할 정복

## 풀이

막대 높이로 Tree를 구성해 각 구간의 최솟값 위치를 트리 형태로 표현했다. 어떤 구간의 최대 직사각형은 "전체 구간 너비 x 최솟값 높이"와 좌우 부분 구간의 답 중 하나이므로 이를 재귀적으로 계산했다.

## 복잡도

- 시간: `O(N log N)`
- 공간: `O(N)`

## 한 줄 회고

최솟값 기준 분할이 바로 이 문제의 핵심 구조다. 해당 구현은 2023년에 한 구현으로 플레 이상 문제중에서 유일하게 c로 구현하였다.
