# 행성 터널

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/2887)

## 사용 알고리즘

- 최소 스패닝 트리
- 정렬

## 풀이

모든 행성 쌍을 간선으로 만들면 너무 많으므로, `x`, `y`, `z` 각각 정렬했을 때 인접한 행성끼리만 후보 간선으로 연결했다. 이렇게 줄인 간선들만으로도 최적 MST를 만들 수 있으므로, 그 그래프에서 MST를 구해 답을 계산했다.

## 복잡도

- 시간: `O(N log N)`
- 공간: `O(N)`

## 한 줄 회고

좌표별 인접 행성만 봐도 충분하다는 점이 핵심 아이디어다.
