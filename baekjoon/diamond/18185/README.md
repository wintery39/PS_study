# 라면 사기 (Small)

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/18185)

## 사용 알고리즘

- 그리디

## 풀이

연속한 2개 공장, 3개 공장을 묶어 사는 것이 단품보다 유리하므로 왼쪽부터 보면서 최적의 묶음을 먼저 처리했다. 특히 `A[i+1] > A[i+2]`인 경우에는 2개 묶음을 먼저 써서 뒤에서 3개 묶음을 더 많이 활용하도록 만들었다.

## 복잡도

- 시간: `O(N)`
- 공간: `O(1)`

## 한 줄 회고

그리디 알고리즘을 구현할 때 조건에 따라 다른 작업을 수행하도록 할 수 있다는 것을 깨닫는 계기가 되었다.
