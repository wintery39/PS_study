# Ax+Bsin(x)=C

## 링크

[문제 바로가기](https://www.acmicpc.net/problem/13705)

## 사용 알고리즘

- 수학
- 고정밀 연산

## 풀이

`Decimal`로 정밀도를 확보한 뒤, 함수 `Ax + Bsin(x)`가 단조 증가한다는 점을 이용해 해가 있는 구간에서 반복문을 통해 탐색을 진행하였다. 시간복잡도를 엄격하게 보는 문제가 아니여서 이분탐색을 진행하지는 않았다. `sin`, `cos`는 급수와 반각 공식을 이용해 직접 계산했다.

## 복잡도

- 시간: `O(I)`
- 공간: `O(1)`

## 한 줄 회고

고정밀 연산을 위해선 Python의 Decimal을 이용하거나 소수값을 정수형으로 바꾸는게 효율적이라는 것을 알게 됐다.
