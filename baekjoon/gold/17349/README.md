# 1루수가 누구야 

## 링크
[문제 바로가기](https://www.acmicpc.net/problem/17349)

## 사용 알고리즘
- 브루트포스

## 풀이
1루수가 아니라고 정해지지 않은 모든 사람은 1루수가 될 수 있다는 의미므로 True로 나타내고 1루수가 확실히 아닌 사람만 False로 나타낸다. 이 때, 'n번이 1루수다' 라는 의미는 'n번을 제외한 모든 이들이 1루수가 아니다' 라고 표현할 수 있으므로 n번을 제외한 모든 이를 False로 표현하였다.

## 복잡도
- 시간: O(1) 
- 공간: O(1)
