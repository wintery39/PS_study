# 꼬인 전깃줄

## 링크
[문제 바로가기](https://www.acmicpc.net/problem/1365)

## 사용 알고리즘
- LIS(Longest Incresing Subsequence)

## 풀이
문제에서 돌려서 표현하고 있지만 가장 긴 증가하는 부분수열의 길이를 구하고 나머지 수들을 모두 제거한다고 생각하면 쉽게 풀 수 있다.

## 복잡도
- 시간: O(n logn) 
- 공간: O(n)

rust에 대해 연습하는 시간이었다. 특히 rust에서 재귀함수를 사용하는 것에 대해 익숙해지는 시간이었다.
